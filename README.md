# DSA-Solutions

This repository contains **30+ Data Structures and Algorithms programs**
implemented using the **C programming language**.

The programs include a mix of **menu-driven** and **simple** 
implementations covering major DSA topics.

---

##  Topics Covered

- Arrays
- Strings
- Stack
- Queue
- Linked List
- Searching
- Sorting
- Trees
- Graphs
- Recursion
- Matrix Operations
